## 🌍 О проекте

**КАРМА ПУСТОШИ** — уникальный жанр Rogue-Anarchy, где каждое действие формирует судьбу игрока. Система кармы влияет на лут, отношение NPC и доступ к возможностям. Нет спавна — каждый игрок появляется в случайной точке мира и начинает выживать с нуля.

### Ключевые особенности
- ⚖️ **Система Кармы** — герои получают x1.5 лут, злодеи — x0.7
- 🎲 **Случайный спавн** — появление в радиусе 500-3000 блоков от центра
- 📡 **Координаты лут-зон** — регулярное вещание координат NPC-зон
- 🏴 **Анархия** — PvP включён, приваты ограничены
- 💀 **5 тиров NPC** — от слабых бродяг до рейд-боссов

## 🚀 Установка

### Требования
- Linux x86_64
- PHP 8.4 (входит в комплект `bin/`)
- PocketMine-MP 5.30.0 (`PocketMine-MP.phar`)

### Запуск
```bash
# 1. Скачайте репозиторий
git clone https://github.com/kevinriverrrr-sudo/SadCraft-KARMA-PUSTOSHI.git
cd SadCraft-KARMA-PUSTOSHI

# 2. Скачайте PocketMine-MP.phar и PHP бинарники
# (не включены в репозиторий из-за размера)
# Phar: https://github.com/pmmp/PocketMine-MP/releases
# PHP: https://github.com/pmmp/PHP-Binaries/releases

# 3. Запустите сервер
chmod +x start.sh
./start.sh
```

### Структура проекта
```
mc-server/
├── start.sh                    # Скрипт запуска
├── server.properties           # Конфигурация сервера
```

## ⚖️ Система Кармы

| Карма | Статус | Множитель лута |
|-------|--------|----------------|
| ≥ 50 | 🟢 Герой | x1.5 |
| 10 — 49 | 🔵 Мирный | x1.0 |
| -49 — 9 | 🟡 Нейтральный | x0.9 |
| ≤ -50 | 🔴 Злодей | x0.7 |

**Способы получения кармы:**
- Убийство NPC: от +1 до +25 (зависит от тира)
- Убийство игрока: -15
- Смерть от игрока: -5
